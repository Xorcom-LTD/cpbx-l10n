bulk_modification.bulk-dialoptions.tooltip = Vous pouvez inclure plusieurs options d'appels pour les appels sortants, comme par exemple forcer l'identication vocale de l'appelant, définir des sonneries distincites, etc. Pour connaitre toutes les options disponibles, saisir la commande "Core show application" dans le CLI Asterisk. Si vous souhaitez définir des options multiples, saisissez les les unes à cotés des autres. Par exemple si vous voulez utiliser les options "m" et "H" il vous suffit de saisir "mH".
