dhcpsettings.tftp.tooltip = La Opción 66 asigna a los teléfonos IP una dirección URL para el aprovisionamiento de configuración. El Endpoint Manager (administrador de terminales) proporciona la información de configuración a los teléfonos IP en respuesta a una solicitud HTTP. El formato de la URL de la solicitud, en este caso se ve como http://[PBX-IP-address]/xepm-provision/. Si se define la dirección IP del PBX o el nombre de host en el campo Opción 66 y se selecciona la casilla "Uso de Endpoint Manager", a continuación el formato correcto de la URL se construirá de forma automática. Si tiene archivos de configuración de telefonía IP ya preparados ubicados en el directorio /tftpboot, entonces deberá incluir la dirección IP del PBX o el nombre de host en el campo Opción 66 y desactivar la casilla "Uso de Endpoint Manager".
