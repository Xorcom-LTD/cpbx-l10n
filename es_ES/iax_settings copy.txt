iax_settings.maxcallnumbers.tooltip = La opción 'maxcallnumbers' limita la cantidad de números de llamada permitidos para cada dirección IP remota individual. Una vez que una dirección IP alcanza su límite de conexiones, no se permitirán nuevas conexiones hasta que se cierren las anteriores. Esta opción también se puede utilizar en la definición de un par, pero solo surte efecto para la dirección IP de un par dinámico después de que complete el registro.

iax_settings.maxcallnumbers_nonvalidated.tooltip = La opción 'maxcallnumbers_nonvalidated' se utiliza para establecer la cantidad combinada de números de llamada que se pueden asignar para conexiones donde se ha desactivado la validación del token de llamada. A diferencia de la opción 'maxcallnumbers', este límite no es independiente para cada dirección IP individual. Cualquier conexión que resulte en la asignación de un número de llamada no validado por token contribuye a este límite.
