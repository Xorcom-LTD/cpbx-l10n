cloudphone.push.tooltip = CUAD Mobile devices will by default receive push notifications from a remote server that maintains connectivity to the PBX on behalf of the device. This is done to reduce battery usage on CUAD Mobile devices but requires the PBX to be accessible externally. Disabling push notifications will allow the CUAD Mobile device to make and receive calls via the local network but will reduce battery life. Los dispositivos CUAD Móvil recibirán de forma predeterminada notificaciones automáticas de un servidor remoto que mantiene la conectividad con el Conmutador en nombre del dispositivo. Esto se hace para reducir el uso de la batería en los dispositivos móviles CUAD pero requiere que el Conmutador sea accesible externamente. La desactivación de las notificaciones automáticas permitirá que el dispositivo móvil CUAD realice y reciba llamadas a través de la red local, pero reducirá la duración de la batería.
