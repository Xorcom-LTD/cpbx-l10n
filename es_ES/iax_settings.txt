iax_settings.maxcallnumbers = Máximo Número de Llamadas
iax_settings.maxcallnumbers.tooltip = La opción 'maxcallnumbers' limita la cantidad de llamadas permitidas para cada dirección IP remota individual. Una vez que una dirección IP llega a su límite de número de llamadas, no hay más conexiones nuevas hasta que las anteriores se cierren. Esta opción se puede utilizar también en una definición de peer, pero sólo tiene efecto para la IP de un interlocutor dinámico después que se completa el registro.
iax_settings.maxcallnumbers_nonvalidated = Máximo Número de Llamadas No Validadas
iax_settings.maxcallnumbers_nonvalidated.tooltip = El 'maxcallnumbers_nonvalidated' se utiliza para establecer el número total de números de llamada que se pueden asignar para las conexiones de llamadas, donde el token de validación ha sido desactivado. A diferencia de la opción 'maxcallnumbers', este límite no está separado para cada dirección IP individual. Cualquier conexión que resulte en un número de llamada validado  por un token no relacionado a llamadas, contribuye a este límite.
